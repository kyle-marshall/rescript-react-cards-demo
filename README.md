# ReScript Project Template

The only official ReScript starter template.

Followed the modifications outlined here:
https://dusty.phillips.codes/2021/09/30/fast-bare-rescript-react/

## Installation

```sh
npm install
```

## Build

- Build: `npm run build`
- Clean: `npm run clean`
- Build & watch: `npm run start`

## Run

```sh
node src/Demo.bs.js
```
